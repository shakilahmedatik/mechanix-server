# ✨MechaniX | Car Repair Services & Auto Mechanic✨
> Complete MERN-STACK Project 

This project is live at- [MechaniX🚀](https://mechanix.netlify.app/).

## Build Stack:

In this project I've used:

 #### ✔️ `React - Front-End`

 #### ✔️  `Node.js - Back-End`

 #### ✔️ `MongoDB - Database`

 #### ✔️ `Firebase - Authentication`
